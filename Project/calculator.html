JavaScript Calculator

The JavaScript Calculator is a responsive web application built using HTML, CSS, and JavaScript. It performs basic arithmetic operations and demonstrates DOM manipulation and event handling.

🚀 View Live Calculator 💻 View Source Code

1. User Interface

The calculator interface consists of number buttons, operator buttons, and a display screen. The design is clean and responsive.

2. Functionality

When a user clicks a button, JavaScript captures the value using event listeners. The input is displayed on the screen and evaluated using logic functions.

3. Features

  • Basic arithmetic operations (+, -, *, /)
  • Clear button
  • Percentage calculation
  • Power (x^y) operation
  • Responsive design

Technologies Used

HTML, CSS, JavaScript
